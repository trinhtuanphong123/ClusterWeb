# pipelines — Data Processing Pipelines

Python pipeline packages for the worker service on AWS EC2.

Handles ingestion, feature engineering, clustering, and historical pattern retrieval.

## Packages

| Package | Purpose |
|---|---|
| `common/` | Shared utilities (DB helpers, config, logging) |
| `ingestion/` | VNStock data ingestion |
| `features/` | Feature engineering (returns, volatility, drawdown, volume, market/sector-relative) |
| `clustering/` | Behavior clustering |
| `retrieval/` | Historical pattern retrieval |
